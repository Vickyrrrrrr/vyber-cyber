# Vyber: Autonomous Self-Healing Cyber-Range

An autonomous cyber-range dashboard simulating Red Team penetration testing and self-healing Blue Team mitigation within serverless sandboxes.

### Core Stack
* **Backend**: Serverless container orchestration using Modal (`modal.App`).
* **Agents**: OpenCode CLI integration for execution tools.
* **Frontend**: Minimalist dark Gradio Block dashboard.
